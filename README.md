•Developed an AI-powered web app for MRI brain tumor classification across 4 tumor classes using a
dataset of 5,800 MRI images.

•Performed data preprocessing like normalization, resizing and one-hot encoding using NumPy and Pandas

•Trained and Evaluated multiple deep learning models (Custom CNN, EfficientNet, ResNet, DenseNet)
using TensorFlow and Scikit-Learn; selected DenseNet achieving 93 percent accuracy with lowest false
negatives.

•Integrated the model into a full-stack application with Flask REST APIs (backend), SQLite (database),
and HTML/CSS/JavaScript (frontend).
